## 🆘 Quick Problem Solving

### ❌ Something's Broken?
```bash
./scripts/health-check.sh --repair
```

### ❌ Still Broken?
```bash
./scripts/maintenance.sh reset

```

### ❌ Need Help?
1. Check [`scripts/TROUBLESHOOTING.md`](scripts/TROUBLESHOOTING.md)
2. Run `./scripts/health-check.sh`
